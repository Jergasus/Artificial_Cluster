Descripción detallada

Representa un Procesador.

Dispone de dos estados posibles (inicializado / no inicializado); si está inicializado tiene una memoria y unos procesos. En caso contrario, es vacío.

Avanzar t unidades de tiempo en los procesos del procesador.

Precondición
El parámetro implícito está inicializado.
Postcondición
Se avanza t unidades de tiempo en cada proceso. Si el tiempo de un proceso ha finalizado, se elimina del procesador. En caso contrario, los procesos progresan t unidades de tiempo.
